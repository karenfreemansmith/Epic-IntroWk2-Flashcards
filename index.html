JavaScript Flashcards

JavaScript (JS)

The only programming language understood by all modern web-browsers! Used in front-end development to create interactivity in web sites.

Operators

An operator is a special character (or characters) that indicates an action to be performed, such as +, -, *, /, and %.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.
Always use the var keyword when initializing variables in JavaScript.

Variable naming conventions

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

An argument is what you pass into a function or method.
Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

A parameter is a variable that's assigned to the argument. Above, number1 and number2 are parameters, and 1 and 2 are arguments.

Return

Once called, many functions will provide a result of some kind. This is known as returning, and the result is often referred to as a return value.

Chaining methods

Chaining metthods is when you call a method directly on the return value of another method.

Strings

'String' is a JavaScript data type that represents the exact text of whatever is enclosed in the quotes

Booleans

true and false are booleans. They are not strings - they simply represent being true or false.

Undefined

a primitive data type in JavaScript indicating no value has been assigned.

NaN

NaN stands for "Not a Number" and it is considered a special number in JavaScript

Escape

An escape character allows us to use special characters inside of a string. "\" is the escape character in JavaScript

Alert

Alert is a built-in methos of JavaScript that pops up a message box with a message that can only be read and dismissed.

Comments

Anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

jQuery is a JavaScript library that contains functions for many common tasks and can save many lines of code when we use it.

Attributes

Attributes are additional information about HTML elemetns (such as name or id). JavaScript can access attributes from HTML elements through the DOM.
